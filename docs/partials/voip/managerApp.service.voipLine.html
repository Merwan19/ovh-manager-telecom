voipLine
service in module managerApp

Description

Describe a service that manage sip, mgcp, plugAndFax, ... feature types (all featureTypes that are not fax and freefax for line services).

This service will manage API calls to /telephony/{billingAccount}/line/{serviceName} (see telephony line APIs)
