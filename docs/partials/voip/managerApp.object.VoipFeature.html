VoipFeature
object in module managerApp

Description

Factory that describes a voip feature (e.g. sip, mgcp, contactCenterSolution, ...). This is a base for VoipLineFeature factory and future VoipAliasFeature factory.

Should not directly instanciated.

Usage

VoipFeature(options);

Parameters

ParamTypeDetails
optionsObject

Shared (required) options for creating a new VoipFeature instance.

PropertyTypeDetails
billingAccountString

The billing account of the feature.

serviceNameString

The service name of the service linked to the feature.

featureTypeString

The type of the feature (sip, mgcp, contactCenterSolution, ...).
