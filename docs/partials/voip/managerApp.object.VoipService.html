VoipService
object in module managerApp

Description

Factory that describes a service with attributes returned by /telephony/{billingAccount}/service/{serviceName} API.

See telephony.TelephonyService enum for available properties.

Usage

VoipService(options);

Parameters

ParamTypeDetails
optionsObject

Options required for creating a new instance of VoipService (see telephony.TelephonyService enum for availables options properties).

Note that billingAccount and serviceName options are mandatory.

Properties

  • getDisplayedName

    Get the displayed name of the service.

    Returns
    String

    The displayedName of the service (the description if provided, the serviceName value otherwise).

  • isDescriptionSameAsServiceName

    Helper that check if the description is the same as serviceName.

    Returns
    Boolean

    true if description is the same that serviceName.
