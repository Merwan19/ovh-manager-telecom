VoipBillingAccount
object in module managerApp

Description

Factory that describes a billingAccount with attributes returned by /telephony/{billingAccount} API.

See telephony.BillingAccount enum for available properties.

Others "custom properties" are listed in Properties section bellow.

Usage

VoipBillingAccount(options);

Parameters

ParamTypeDetails
optionsObject

Options required for creating a new instance of VoipBillingAccount (see telephony.BillingAccount enum for availables options properties).

Note that billingAccount option is mandatory.

Methods

  • addService(service)

    Add a service to billing account services list. If service already exists in list (meaning that a service with serviceName option is already in the list), the existing service will be replaced by a service with the new options passed in arguments.

    Parameters
    ParamTypeDetails
    serviceObjectVoipService

    The options for creating a new VoipService instance or a VoipService object.

    Returns
    VoipService

    The VoipService instance added (or replaced) to the services list.

  • addServices(services)

    Add one or many services to the billing account services list.

    Parameters
    ParamTypeDetails
    servicesArray.<Object>Object

    An object or a list of objects with options for adding VoipService instances to billing account services list.

    Returns
    Array.<VoipSercice>

    The list of all the services of the billing account (including added services).

  • getAlias()

    Get the services of the billingAccount that match alias serviceType.

    Returns
    Array.<VoipSercice>

    The list of all services with serviceType alias of the billing account.

  • getDisplayedName()

    Get the displayed name of the billing account.

    Returns
    String

    The displayedName of the billingAccount (the description if provided, the billingAccount value otherwise).

  • getLines()

    Get the services of the billingAccount that match line serviceType.

    Returns
    Array.<VoipSercice>

    The list of all services with serviceType line of the billing account.

Properties

  • services

    Store the services attached to the VoipBillingAccount instance.

    You can populate this services list with addService method or with addServices method.

    Returns
    Array.<VoipService>

    An array of VoipService instances.
