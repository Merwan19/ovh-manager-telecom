VoipLine
object in module managerApp

Description

Inherits from VoipLineFeature.

Factory that describes a line feature with attributes returned by /telephony/{billingAccount}/line/{serviceName} API.

Usage

VoipLine(options);

Parameters

ParamTypeDetails
optionsObject

Options required for creating a new instance of VoipLine (see VoipLineFeature constructor for availables inherited options properties and telephony.Line enum for specific line options properties).

Properties

  • setOptions

    Set the options from telephony.Line enum. This is called by default by the constructor.

    Returns
    VoipLine

    The VoipLine instance with options setted.
